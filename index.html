steer

by Jeremias Menichelli

Script that fires events only when the user changes the scrolling direction.

Scroll down and up again and you will notice that a message is displayed on the page informing in which scrolling point it was fired.

You can download the code from the github repository or download it from bower as steer and npm as steerjs.

Support: Chrome, Firefox, Safari and IE9+
